About Cybersphere

Cybersphere is an innovative and modular security suite developed to strengthen system-level protection by actively monitoring and detecting unauthorized or suspicious activities. The project integrates four core modules—each designed to counter a specific form of security threat—creating a holistic approach to personal cybersecurity.

In today's interconnected world, digital security is more critical than ever. With the rapid advancement of technology, cyber threats have grown in both sophistication and frequency. Individuals and organizations are constantly at risk of falling victim to unauthorized surveillance, phishing attacks, information leakage, and malware.

Security Modules

Screenshot Detector

Detects and alerts on unauthorized screenshot attempts, whether from built-in OS features or third-party tools.

  • Monitors default screenshot directory
  • Recognizes screenshot hotkeys
  • Identifies screenshot processes
  • Supports attack simulation

Folder Detector

Continuously scans sensitive folders for creation or modification of suspicious file types.

  • Detects file creation/modification
  • Checks against suspicious extensions
  • Triggers alerts and auto-removal
  • Dynamic configuration of watch directories

Webcam Detector

Detects real-time unauthorized webcam access, simulates webcam-based attacks, and alerts the user instantly.

  • Monitors webcam access attempts
  • Flags suspicious access
  • Tracks webcam state changes
  • Offers webcam simulation
